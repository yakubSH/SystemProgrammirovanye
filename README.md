 📂 Структура репозитория

- yakubSH — проект или папка (предположительно, по фамилии автора)
- README.md — описание репозитория
- Задание.docx — текст задания
- Лампочка ЛБ2.docx — материалы лабораторной работы №2 по теме «Лампочка»
- Лампочки ЛБ1.docx — материалы лабораторной работы №1 по теме «Лампочки»
- Определения и их значения.docx — документ с определениями терминов
- Отчет сп1.docx — отчёт по практике или работе (СП1)
- Системное программирование.docx — материалы по системному программированию
